# Scalar

# About
Scalar is a warioware inspired game with 5 microgames all having to do with scaling! It was a fun little way for me to get back into raylib and whilst it isnt my best work its still a charming little game to play around with!

# Microgames!
Scalar has 5 total microgames these are:
| Game name | What do I do? |
|-----------|---------------|
|Climb mountain| Alternate between A and D to make the little guy climb up|
|Drop weight| Drop the weight on the other side of the scale to make it balanced! |
| Fit coin | Scale the coin down so that it fits into the piggy bank|
| Guess weight | You are given 3 items and their repective weights. Match the item of the weight to win!
| Inflate balloon | Inflate the balloon so it rises up|


# Building
Should you wish to build it, there is a makefile provided using `make`
